## 2. レイアウト構成

- **最大幅**: 1200px（センタリング）
- **グリッド**: CSS Grid / Flexbox ベース
- **ヘッダー**: スティッキー固定
- **CTAボタン**: 大きめ・センタリング
- **レスポンシブ対応**: モバイル（768px以下）でスタック切替、ハンバーガーメニュー

## 4. UIパターン

- **ナビゲーション**: スティッキー、半透明ホワイト＋blur効果、モバイルはドロワー
- **ボタンスタイル**: 大きめ丸角（`border-radius: 24px`）、矢印アイコン付き
- **カードスタイル**: 薄いオレンジ背景、丸角、シャドウ
- **画像スタイル**: 有機的なblob形状クリッピングで柔らかさを演出
- **見出しスタイル**: 英語大文字メイン＋日本語サブ見出し
- **アニメーション**: ホバー時の色変化・スライド、スクロールでのフェードイン

---

## 5. フォントの雰囲気

- **使用フォント**: `Noto Sans JP`（Google Fonts）
- **見出し**: `font-size: 2rem; font-weight: 700`
- **CTAボタン**: `font-size: 1.25rem; font-weight: 700`
- **本文**: `font-size: 1rem`
- **雰囲気**: 読みやすくモダン、親しみやすいゴシック体
