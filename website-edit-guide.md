## 1. GitHubアカウントの作成

### 手順

1. ブラウザで [https://github.com](https://github.com) を開く

2. 右上の **「Sign up」** ボタンをクリック

3. 以下の情報を入力して登録する
   - メールアドレス
   - パスワード（英数字・記号を含む8文字以上）
   - ユーザー名（半角英数字、後から変更可能）

4. メールが届いたら、メール内のリンクをクリックしてアカウントを認証する

5. ログイン完了

### アカウント作成後にやること（1回だけ）

アカウントを作成したら、**サイト管理者（gitアカウント: jussssss37）に自分のGitHubユーザー名を伝えて、編集権限を付与してもらう**必要があります。
権限がないとファイルを保存できないので、必ず先に連絡してください。

## 2. リポジトリへのアクセス

1. GitHubにログインした状態で、以下のURLを開く
   👉 **[https://github.com/jussssss37/asahi-first-web](https://github.com/jussssss37/asahi-first-web)**

2. 画面にファイルの一覧が表示される（これが「リポジトリ」）

3. フォルダをクリックすると中のファイルが見られる

## 3. ファイルの編集方法

### 基本的な編集手順

1. 編集したいファイルを開く（後述の「よくある修正パターン」を参考に）

2. ファイルの右上にある **鉛筆アイコン（✏️）** をクリック

   ![edit button](https://docs.github.com/assets/cb-47677/mw-1440/images/help/repository/edit-file-edit-button.webp)

   > ※ 鉛筆アイコンが灰色でクリックできない場合は、権限が付与されていません。管理者に連絡してください。

3. テキストが編集できる状態になる。修正したい箇所を変更する

4. 画面右上の緑色の **「Commit changes...」** ボタンをクリック

5. ポップアップが表示されたら、そのまま **「Commit changes」** ボタンをクリックして保存

   > **「Commit message」** 欄には、何を変更したか一言書くと後で分かりやすい（例: `電話番号を更新`）。空欄でも保存できる。

6. 保存完了。数分後にホームページに反映される

## 4. よくある修正パターン

### 会社情報の変更（会社名・住所・電話番号など）

**対象ファイル:**
`asahi-first-web/src/data/company.ts`

👉 [ファイルを開く](https://github.com/jussssss37/asahi-first-web/blob/master/src/data/company.ts)

このファイルに会社の基本情報がまとまっています。

```
name: "株式会社アサヒファースト",     ← 会社名
ceo: "喜多 実輝",                    ← 代表者名
zip: "532-0013",                    ← 郵便番号
address: "大阪府大阪市此花区...",     ← 住所
tel: "08048862577",                 ← 電話番号
founded: "2024年3月",               ← 設立日
capital: "2,000万円",               ← 資本金
services: "労働者派遣事業",          ← 事業内容
licenses: "労働者派遣事業許可番号：派27-305315",  ← 許可番号
```

**修正例:** 電話番号を変えたい場合
`"08048862577"` の部分を新しい電話番号に書き換えるだけ。

> ⚠️ **注意:** `"` （ダブルクォーテーション）と `,`（カンマ）は消さないでください。文字の中身だけ変更してください。

## 5. 注意事項

### やってはいけないこと
- ダブルクォーテーション `"` やカンマ `,` を消す
- `<` `>` `{` `}` などの記号を削除・変更する
- ファイルを丸ごと削除する

### 間違えてしまったら
- 編集画面で **Ctrl+Z**（Mac は Cmd+Z）で元に戻せます
- 保存してしまった後でも、管理者に連絡すれば以前の状態に戻せます（gitの履歴が残るため）

### 反映について
- 保存後、ホームページへの反映には **数分〜10分程度**かかる場合があります
- 反映を確認するにはブラウザのキャッシュをクリアしてから確認してください（Ctrl+Shift+R）
