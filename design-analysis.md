## 2. レイアウト構成

- **最大幅**: `1366px`（カスタムCSS Grid）
- **グリッド**: Bootstrap 5.3.2 ベース、`full / feature / popout / content` の4段階レイアウト
- **ヘッダー**: スティッキー、`padding: 8px 58px`（PC）
- **CTAボタン**: 最大幅 `768px`、センタリング
- **コラム**: 3カラムグリッド（`calc(33.333% - 2rem)` 各幅）
- **フッター**: 4カラムナビ（`calc((100% - 4.5rem) / 4)` 各幅）
- **レスポンシブ**:
  - `767px` 以下: モバイル（右スライドドロワーメニュー、幅80%・青背景）
  - `1365px` 以下: タブレット〜ワイド切替

## 4. UIパターン

### ナビゲーション
- スティッキーヘッダー
- **ガラスモーフィズム**: `rgba(255,255,255,0.7)` + `backdrop-filter: blur(10px)`
- PCでは横並びメニュー。求職者/企業向けナビアイテムには左端に6px幅のカラーボーダー（青・オレンジ）
- モバイルではハンバーガーボタン → 右スライドドロワーメニュー（幅80%、青背景、白テキスト）

### CTAボタン（大型）
- 幅 `768px`、`border-radius: 24px`（大きめ丸角）
- `border: double 6px white`（ダブルボーダー）
- 右端に白い矢印アイコン（Base64埋め込みPNG）
- 求職者向け: 青（`#0098b9`）
- 企業向け: オレンジ（`#FFA500`）
- お問い合わせ: ダークグレー（`#333333`）

### 記事カード
- 3カラムグリッド
- `border: 2px solid #f2f2f2`（薄いグレーボーダー）
- 「NEW」ラベル: `#FF5722` バッジ
- 「詳しく見る」: ホバー時に `#b1dae7` の丸型背景がスライドするマイクロアニメーション

### 画像スタイル
- 記事サムネイル: 四角（角丸なし）にテキストオーバーレイ
- CTA画像: **有機的なblob形状クリッピング**（アメーバ型）で柔らかさを演出

### セクション見出し
- 英語大文字（例: `COLUMN`, `CONTACT`）をメイン見出し
- `#FF5722` の日本語サブ見出しを `::after` 疑似要素で表示するバイリンガル構成

### ページトップボタン
- `width: 60px; height: 60px; border-radius: 50%` の円形固定ボタン
- 右下 45px 位置、背景色 `#0098b9`

## 7. 全体の印象

**「清潔感・信頼感・モダン」を基調とした人材派遣業らしい日本的コーポレートデザイン。**

### ポジティブな特徴
- ティール（`#0098b9`）＋オレンジの補色構成で「誠実さ×活気」のバランスを実現
- 求職者/企業の2ターゲットが**色で明示**され、ユーザーが迷わず導線を選べるUX設計
- ガラスモーフィズムヘッダー、blob型画像クリッピング、ホバーアニメーションなど2020年代のモダントレンドを適切に取り入れている
- フッターのグラデーション（ティール→ライトブルー）が重厚感より軽快さと親しみやすさを演出

### 設計方針
- 派手さより誠実さを優先した「手堅いプロフェッショナルデザイン」
- 製造・軽作業系の求職者と採用企業、両方に配慮した機能的かつ親しみやすい構成
- アニメーション（ローディングスプラッシュ、ホバーインタラクション）が細部まで丁寧に実装
